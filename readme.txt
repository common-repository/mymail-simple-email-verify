== Description ==

> This Plugin requires [MyMail Newsletter Plugin for WordPress](https://evp.to/mymail?utm_campaign=wporg&utm_source=Simple+Email+Verify+for+MyMail)

Verifies your subscribers email addresses. Read more on the [MyMail Blog](https://mymail.newsletter-plugin.com/reduce-false-signups-with-email-verification/?utm_campaign=wporg&utm_source=Simple+Email+Verify+for+MyMail&utm_medium=link)

== Installation ==

1. Upload the entire `mymail-simple-email-verify` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings => Newsletter => Email Verification
